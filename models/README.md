# Models Directory

Place your `.gguf` model files here. They will be copied to `~/WickermanSupport/models/` during installation.

These files are excluded from Git (see `.gitignore`) due to their size.

## Recommended Models

- **Qwen2.5-Coder-14B-Instruct** (Q5_K_M or Q8_0) — coding assistant
- **TinyLlama-1.1B-Chat** (Q4_K_M) — fast test/fallback model
- **all-MiniLM-L6-v2** (f16) — embedding model for RAG memory

Download from HuggingFace or use the built-in Downloader plugin after installation.
